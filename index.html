Guide:

Start - This button will start the stop-watch.

Pause - This button will pause the running stop-watch. You can resume it by clicking again on start button.

Reset - This button will reset the stop-watch at 00:00:00

Save - This button will save the current stop-watch time in the log. You can view it under the "History Log" section below.
